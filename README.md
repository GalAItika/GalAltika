# 🤖 AI English Bot

Telegram-бот для изучения терминов искусственного интеллекта с базой данных, статистикой и ежедневной рассылкой.

## 🚀 Возможности
- Изучение AI-терминов (английский → русский)
- Статистика прогресса (общая + за неделю)
- Добавление новых слов через бота
- Ежедневная рассылка нового слова в 10:00

## 🛠️ Технологии
- Python 3.10+
- `python-telegram-bot` — Telegram API
- `sqlite3` — база данных
- `apscheduler` — ежедневная рассылка
- Google Colab

## 📦 Как запустить
1. Открыть файл `AI_English_bot.ipynb` в Google Colab
2. Вставить свой токен от @BotFather
3. Нажать **Run** (Ctrl+Enter)
4. Написать боту `/start` в Telegram
